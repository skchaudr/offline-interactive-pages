Before any workflow

Leader keys only work in Normal mode. If nothing happens, press Esc first.

SymptomFix
Leader does nothingEsc — you are in Insert or Terminal mode
gf fails on [[link]]Use Enter on the link (markdown buffers only)
Space o l did the wrong thingNormal = outgoing links list. Visual = link selection
Space o n vs Space o don = new note. od = today's daily
Space o q vs Space o soq = find by title. os = search inside note text
