# Rules

Every artifact must:

- work offline
- use semantic HTML
- be touch friendly
- support desktop + tablet
- support dark mode
- contain real educational value
- be easy to edit manually
- avoid unnecessary complexity

Default:

single-file `index.html`

Allowed:

- inline CSS
- inline JS
- localStorage
- SVG
- Canvas

Forbidden:

- CDNs
- npm dependencies
- frameworks by default
- analytics
- placeholders
