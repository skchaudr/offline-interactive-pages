# Style Guide

Design goals:

- calm
- tactile
- premium
- educational
- interactive
- clean information density

Patterns:

- cards
- tabs
- accordions
- diagrams
- interactive demos
- keyboard chips
- sticky navigation
- collapsible sections
- progressive reveal

Typography:

- system font stack
- large readable headings
- comfortable spacing

Visual feel:

Apple docs × Linear × Stripe docs × NotebookLM
